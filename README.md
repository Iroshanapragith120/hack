A powerful terminal-based satellite tracking animation designed to simulate a high-level hacking interface. This project features a rotating global scan, detailed Sri Lanka map zoom, and real-time data extraction simulation.

---

## 💀 FEATURES

* **Global Satellite Scan:** Simulated rotating Earth scan using `blessed-contrib`.
* **High-Detail Sri Lanka Map:** Precision ASCII art representing the Sri Lankan region.
* **Real-time Logs:** Scrolling hacker-style system logs and firewall bypass alerts.
* **Signal Gauge:** Dynamic signal strength progress bar.
* **Target Pinpointing:** Automatic coordinate locking on target region.
